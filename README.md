# BMI Calculator

A simple web application for calculating Body Mass Index (BMI).

## Features
- Calculate BMI based on weight and height.
- Categorize BMI into Underweight, Normal weight, Overweight, or Obesity.
- Input validation with error messages for invalid data.

## How to Run
1. **Install Node.js** (if not already installed).
2. How to setup:
   ```bash

   cd bmi-calculator
